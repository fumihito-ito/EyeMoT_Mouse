# 仕様書 — TobiiEyeMouse

## 概要

Tobii Eye Tracker 5 の視線データでマウスカーソルを制御する Windows WPF アプリ。
Tobii Experience の DLL を動的ロードするため SDK のインストール不要。

---

## 操作モード

| モード | 説明 |
|--------|------|
| 視線UIモード | WPFオーバーレイで視線カーソルを描画し、注視でUIを操作。デフォルト。 |
| マウスモード | Win32 `SetCursorPos` でシステムカーソルを直接移動。 |

- **注視操作枠（Dwell Click）は視線UIモード時に無効化**される。

---

## 注視クリック（Dwell Click）— マウスモード専用

| パラメータ | 範囲 | デフォルト | 単位 |
|-----------|------|-----------|------|
| クリック種別 | なし / クリック / ダブルクリック / ドラッグ | クリック | — |
| 注視時間 | 200〜3000 | 2000 | ms |
| 判定半径 | 10〜150 | 50 | px |
| クールダウン | 100〜2000 | 1000 | ms |

### ドラッグ動作
1. 注視時間が経過するとドラッグ開始（`MOUSEEVENTF_LEFTDOWN`）
2. 次の注視完了でドロップ（`MOUSEEVENTF_LEFTUP`）し、クリックモードに自動復帰
3. ドラッグ中は判定半径を 1.5 倍に拡大してドロップしやすくする

---

## ブレ補正・フィルタ設定

| パラメータ | 範囲 | デフォルト | 説明 |
|-----------|------|-----------|------|
| 応答性（α） | 5〜100% | 90% | EMA の重み。高いほど追従が速い |
| デッドゾーン | 0〜60 px | 0 | この距離未満の移動を無視 |
| 最大速度 | 500〜15000 | 5000 | フレームあたりの最大移動量 (px/s) |
| ダイレクトモード | ON/OFF | OFF | フィルタをすべてバイパス。最速応答 |

### EMA フィルタ計算式
```
emaX = α × rawX + (1 - α) × emaX
outX = clamp(outX + Δx, 0, screenWidth - 1)
```
デッドゾーン内では `outX` を更新しない。最大速度は `dt × MaxSpeedPx` でクランプ。

---

## カーソル設定

| 形状 | 記号 | 説明 |
|------|------|------|
| なし | ∅ | カーソル非表示 |
| 十字 | ✚ | 中心に赤ドット付き十字線 |
| 円 | ● | 半透明塗りの円 |
| ひし形 | ◆ | 半透明塗りのひし形 |
| リング | ◎ | 中空リング＋中心赤ドット |
| ドット | ⚫ | 小さな黄色ドット |

- サイズ：20〜200 px（デフォルト 100 px）
- ドラッグ中はオレンジ系に配色変更

---

## スクロール設定

| パラメータ | 範囲 | デフォルト | 説明 |
|-----------|------|-----------|------|
| 有効/無効 | ON/OFF | OFF | 画面上下端でのスクロール |
| エッジ判定幅 | 50〜400 px | 100 px | 上下それぞれのスクロール発火ゾーン幅 |
| スクロール速度 | 1〜20 | 3 | 1回の wheel イベント量（×120）|

- 最大発火レート: 20回/秒（50 ms インターバル）
- 上端 → スクロールアップ、下端 → スクロールダウン

---

## GazeOverlay（視線オーバーレイ）

全画面透明 WPF ウィンドウ（`WS_EX_TRANSPARENT` でクリック透過）。
60 fps（16 ms タイマー）でレンダリング。

### 描画レイヤー（前から順）
1. スクロール領域（半透明青色帯）
2. 一時停止中の左上コーナーインジケーター（注視で再開）
3. ドラッグ中の視覚フィードバック（開始点マーカー＋接続線＋ラベル）
4. 注視進捗アーク（ブルー通常／オレンジドラッグ）
5. 視線カーソル

### 停止中の Dim モード
トラッキング停止中は半透明グレーのドットで視線位置だけ表示する。

---

## MiniPanelWindow（最小化パネル）

- メインウィンドウ最小化時に画面上部に表示
- 注視/ホバーでプログレスバーが進み、完了するとメインウィンドウを復元
- `Topmost` / `ShowInTaskbar=False` / 透明背景

---

## グローバルキーボードショートカット

| キー | 動作 |
|------|------|
| F5 | トラッキング 開始 / 停止 |
| F6 | 一時停止 / 再開 |
| Esc | アプリ終了 |
| 設定可能キー | 注視クリックの即時発火（`TriggerClick`） |

ウィンドウ非アクティブでも動作（グローバルキーフック）。

---

## 一時停止の再開方法

一時停止中は左上コーナー（150 × 150 px）に注視するとプログレスが進み、完了で再開。

---

## 設定の永続化

- パス: `%LOCALAPPDATA%\TobiiEyeMouse\settings.json`
- アプリ起動時にロード、変更時に即時保存
- 保存項目: 操作モード、フィルタ全設定、カーソル設定、注視クリック設定、スクロール設定

---

## アーキテクチャ

```
MainWindow
├── TobiiStreamEngine   視線DLL動的ロード・データ受信
├── GazeFilter          EMAフィルタ・デッドゾーン・最大速度
├── DwellClicker        注視クリック判定・ドラッグ管理
├── GazeScroller        画面端スクロール
├── GazeOverlay         全画面透明オーバーレイ描画
├── MiniPanelWindow     最小化パネル
├── GlobalKeyHook       グローバルキーフック
└── AppSettings         JSON設定管理
```

---

## 動作要件

- Windows 10 / 11
- Tobii Eye Tracker 5（USB接続）
- Tobii Experience インストール済み
- .NET 8 以上
